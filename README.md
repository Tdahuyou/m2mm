# m2mm

- m2mm（markdown to markmap）是一款将无序列表形式的 markdown 内容转为 markmap 思维导图格式的工具。
- 转换结果是只读的，主要作为预览用。
- 编写 m2mm 的初衷是为了让自己的笔记大纲能够以思维导图的形式来呈现，以便笔记大纲的预览。

## 🔗 在线访问

- https://m2mm.tdahuyou.cn/
- 可在电脑、平板、手机上访问。
